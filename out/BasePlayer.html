Class: BasePlayer

BasePlayer()

Base class that will be extended by a specific healing class (e.g. Paladin) Contains common methods that are used by all healers such as initialization of spellQueues, mana updating helper functions, etc Class-specific logic should be implemented in the respective child classes, such as castSpell and subtractMana After create a player object, run createRngThresholds and createSpellQueue to complete the initialization.
